A tool to generate random password.

## Usage ##

    usage: passtk.py [-h] [-l LEVEL] [-n LENGTH]

    A tool to generate random password.

    optional arguments:
      -h, --help            show this help message and exit
      -l LEVEL, --level LEVEL
                            The level(1-3, default is 2) of password, higher is
                            complex
      -n LENGTH, --length LENGTH
                            The length of password
